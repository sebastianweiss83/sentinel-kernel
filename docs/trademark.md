# Trademark

## Current state

No trademark is currently registered for the Sentinel name, logo, or
associated marks. The project is Apache 2.0 licensed and has no
commercial trademark holder.

## Intended ownership

The Sentinel name is intended to be held by the governing foundation
once established, not by any commercial entity. The current plan is
transfer to Linux Foundation Europe once the project meets the
stewardship requirements (target v2.0 — see
[`docs/lf-europe-application/README.md`](lf-europe-application/README.md)).

## Why no trademark yet

Registering a trademark under a commercial entity now would create
exactly the governance problem Sentinel exists to solve: a single
commercial holder with discretionary licensing power over an
infrastructure-critical component. The trademark will be registered
only after a neutral foundation is in place to hold it on behalf of
the community.

## Use of the name

Until foundation transfer, the Sentinel name may be used freely in:

- Technical documentation, articles, and academic work
- Integrations, ports, and derivative open-source work under Apache 2.0
- Compatibility claims ("compatible with Sentinel", "uses Sentinel")

The name may **not** be used to imply endorsement by the Sentinel
project of any commercial product, service, or certification without
explicit written permission from the project maintainers.

## Updates

This document will be updated when the foundation is formally
constituted and trademark registration is initiated. See
[`GOVERNANCE.md`](../GOVERNANCE.md) for the current governance model.
